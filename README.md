## 👨‍💻 About Me

Hello! I am an engineering student deeply enthusiastic about the future of computing systems. I believe that true breakthroughs in **Artificial Intelligence** don't just happen within software algorithms—they rely heavily on highly efficient, purpose-built semiconductor architectures.

My ultimate goal is to become a *Full-Stack Hardware-Software Engineer*—understanding how intelligent models learn at the algorithm level (using Python) and how to accelerate their execution right down to the silicon (using VHDL/Verilog).

*   🔭 **Current Focus:** Digital system design, AI model optimization, and hardware accelerators.
*   🌱 **Currently Learning:** Chip Design, FPGA workflows, and Machine Learning foundations.
*   ⚡ **Fun Fact:** I treat designing logic gates like solving an intricate, high-stakes puzzle!
